CSS Helper Classes

List reset (ul/ol)
Setzt alle Basisformatierungen von Listen zurück.

class="list"

Emmet: ul.list>li*4>a[href="#"]{Ein Link}

Ergebnis:

  • Ein Link
  • Ein Link
  • Ein Link
  • Ein Link

Horizontale Liste (ul/ol)
Richtet die Liste horizontal aus.

class="list list--horizontal"

Emmet: ul.list.list--horizontal>li*4>a[href="#"]{Ein Link}

Ergebnis:

  • Ein Link
  • Ein Link
  • Ein Link
  • Ein Link

Horizontale Liste, zentriert (ul/ol)
Muss in einem Block-Element mit "clearfix" sein!

class="list list--horizontal list--center"

Emmet: nav.clearfix>ul.list.list--horizontal.list--center>li*4>a[href="#"]{Ein Link}

Ergebnis:

  • Ein Link
  • Ein Link
  • Ein Link
  • Ein Link

Horizontale Liste mit Trenner (ul/ol)
Der Trenner kann über die "content: ...;" geändert werden.

class="list list--horizontal list--divider"

Emmet: ul.list.list--horizontal.list--divider>li*4>a[href="#"]{Ein Link}

Ergebnis:

  • Ein Link
  • Ein Link
  • Ein Link
  • Ein Link

Faded Image
Dunkelt ein Bild ab, damit Text darauf besser lesbar ist. Die Klasse kann nicht auf img-Tags angewendet werden.

class="faded"

Emmet: .faded

Ergebnis:

Broken Image
Diese Klasse hübscht kaputte Bilder auf. Funktioniert nur in Chrome, in anderen Browsern werden kaputte Bilder ausgeblendet.

class="broken"

Emmet: img.broken

Ergebnis:

Clean Button
Setzt alle Formatierungen eines Buttons zurück, um diesen z.B. wie einen Link aussehen zu lassen.

class="btn-clean"

Emmet: button.btn-clean

Ergebnis:

Custom Checkbox/Radiobutton
Ermöglicht den Austausch des Standard-Icons von Checkboxen/Radiobuttons.

class="custom-control custom-control--radio"
class="custom-control custom-control--checkbox"

Emmet: tl;dr

Ergebnis:

Radiobuttons
Checkboxen
